# Privacy Policy

## 1. Overview
Whatte collects and processes personal data related to physical activity and health metrics to provide training analysis and recommendations.

## 2. Data we collect
- Training data (e.g. cycling activities from Strava)
- Health data (e.g. heart rate, HRV, sleep, weight from HealthKit)
- User-provided data (e.g. FTP, profile settings)

## 3. How data is used
Data is used to:
- calculate training load and readiness
- generate recommendations
- provide analytics and insights

## 4. Data storage
All data is stored on infrastructure controlled by the project owner.

## 5. Data sharing
Data is not shared with third parties.

## 6. User control
Users can request deletion of their data.

## 7. Disclaimer
This project is not a medical service and does not provide medical advice.
